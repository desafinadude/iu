## Accessibility Scores

### Material You
- ✅ WCAG AA compliant
- ✅ All contrasts ≥ 4.5:1
- ✅ Focus indicators visible
- ✅ Clean, readable fonts

### Comic Book
- ✅ WCAG AA compliant at 17px+
- ✅ High contrast (black on white)
- ⚠️ Wobbly borders may be disorienting
- ✅ Thicker borders improve visibility

### Vaporwave
- ⚠️ Dark mode requires careful contrast
- ✅ Neon colors provide high contrast
- ⚠️ Scanlines may trigger vestibular issues
- ✅ Monospace improves dyslexia readability
- 💡 Add `prefers-reduced-motion` support

## Implementation Notes

All themes use the same HTML structure:
```jsx
<Card>
  <CardHeader>
    <CardTitle>Title</CardTitle>
    <CardDescription>Description</CardDescription>
  </CardHeader>
  <CardContent>
    Content here
  </CardContent>
</Card>
```

The theme changes **only** the visual appearance via CSS custom properties. No component refactoring needed!
